# ShiftMate 📅

A full-stack workforce scheduling platform that helps businesses manage employee shifts, availability, and assignments from one dashboard.

🔗 **[Live Demo](https://shiftmate-eilz.vercel.app/login)**

## ✨ Features

- **Employee Scheduling** — Create and assign shifts with availability management
- **Role-Based Dashboards** — Separate views and permissions for managers and employees
- **Authentication** — Secure JWT-based login system
- **REST API** — Scalable backend endpoints handling all shift and user data
- **Responsive Design** — Works on desktop, tablet, and mobile
